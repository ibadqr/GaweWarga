# 🏛️ GaweWarga — Forum Digital RW 02

**GaweWarga** adalah platform forum digital berbasis web yang dirancang khusus untuk memfasilitasi komunikasi, koordinasi, dan penyebaran informasi secara transparan di lingkup Rukun Warga (RW 02). Dengan antarmuka modern layaknya media sosial premium, platform ini mempermudah pengurus dan warga untuk saling berinteraksi secara kondusif.

---

## ✨ Fitur Utama

*   **🔒 Autentikasi & Proteksi Halaman**: Sistem login aman untuk membatasi akses hanya bagi warga terdaftar.
*   **📢 Linimasa Warga Dinamis**: Fitur berbagi informasi, agenda, maupun keluhan secara terpusat.
*   **🏷️ Kategorisasi Info**: Pengelompokan postingan berdasarkan label khusus (Umum, Acara Warga, Kabar Duka, Info Keamanan, dan Aspirasi/Keluhan).
*   **❤️ Interaksi Sosial Modern**: Fitur menyukai (*Likes*) dan membalas komentar (*Comments*) secara langsung dengan antarmuka yang responsif.
*   **🛠️ Manajemen Konten Terkendali**: Hak akses penuh (Ubah/Hapus) postingan khusus bagi warga pemilik konten.
*   **🔔 Notifikasi Terintegrasi**: Sistem pop-up alert yang elegan menggunakan *SweetAlert2* untuk setiap aksi sukses/gagal.

---

## 🛠️ Teknologi yang Digunakan

*   **Backend / Logika**: PHP Native dengan Driver **PDO (PHP Data Objects)** untuk keamanan *SQL Injection*.
*   **Database Engine**: MySQL.
*   **Frontend Framework**: Tailwind CSS (Mobile-first responsive design).
*   **Ikon & Antarmuka**: Font Awesome Pro Icons & Google Fonts Inter.
*   **Komponen Alert**: SweetAlert2.
